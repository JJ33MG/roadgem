# ROADGEM VISION

## Ons Waarom

Reizen is niet langer over het checken van bekende bezienswaardigheden. Het is over authenticiteit, ontdekking en momenten die je leven veranderen. Maar het perfecte moment, de verborgen winbar, dat restaurant dat locals aanbevelen—die vind je nooit met standaard reisgidsen.

**RoadGem**: een AI-intelligente reisassistent die je naar de momenten brengt die je echt wil beleven.

---

## Wat We Maken

Een intelligente road trip planner die:

- **Hidden gems ontdekt** via AI-analyse van authenticiteit, waardering en verborgen waarde
- **Hyperlocale inzichten genereert** door web scraping, Google Places, en real-time community data
- **Persoonlijke itineraires creëert** gebaseerd op jouw voorkeur, budget, en reistempo
- **Affiliate-inkomsten genereert** door naadloze integratie met booking platforms, restaurants, activiteiten
- **Visueel premium voelt** met een editorial design aesthetic (dark navy, mercury-blue accent #af50ff, curation-first UX)

---

## Hoe We Het Doen

**Tech Stack**:
- Frontend: React + TypeScript + Vite + Tailwind CSS
- Backend: Node.js + Express API
- Intelligence: Claude API (claude-sonnet-4-6) + multi-agent orchestration
- Database: SQLite via Prisma ORM (→ PostgreSQL in productie)
- Deployment: Vercel (frontend) + Railway (backend) — nog niet gedeployed

**AI Architecture**:
- Route Planning Agent: Itineraires genereren via Claude
- Discovery Agent: Hidden gems identificeren via Tavily web search + Google Places
- Content Agent: Visuele en geschreven content genereren
- Monetization Agent: Affiliate links injecteren en conversies tracken

**Design Philosophy**:
- Editorial, niet corporate
- Curation-first (geen info-overload)
- Premium, niet makkelijk
- Donker, elegant, intentioneel
- Kleurpalet: deep-space #090909, mercury-blue #af50ff, starlight #f7f9fa, silver #828384

## De Drie Pijlers

### 1. Ontdekking
Gebruikers willen niet touristische hotspots. Ze willen:
- De beste lokale koffie in Lyon
- Een verborgen wandeling in de Toscaanse heuvels
- Een rustig dorp waar kunstenaars leven
- Restaurants waar locals eten, niet toeristen

RoadGem vindt deze door AI-signalen te combineren: Google Places ratings, local reviews, Tavily web research, Reddit discussions.

### 2. Personalisatie
Niet elke reiziger is hetzelfde.
- Budget backpackers willen andere ervaringen dan luxury travellers
- Solo reizigers plannen anders dan groepen
- Kustvakantie ≠ bergwandelen

RoadGem leert jouw reisvoorkeur en genereert routes die voor jou perfect zijn.

### 3. Monetisatie
Via affiliate partnerships — gebruikers betalen niks, platforms betalen commissie:
- Hotel- en vakantiehuisboeking: Airbnb, Booking.com
- Restaurantreservaties: TheFork, Michelin guides
- Activiteiten en tours: Viator, GetYourGuide
- Reisverzeking, autohuur, andere services
